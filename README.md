## 🚀 MediaHub 快速一瞥

```bash
git clone https://github.com/kongbaiming/mediahub.git
cd mediahub && cp .env.example .env
docker compose up -d --build
# CMS :8080  ·  Web Player :8081  ·  API :3000
```

最新版本：[**v0.3.0**](https://github.com/kongbaiming/mediahub/releases/tag/v0.3.0) — 猜你喜欢 · Feed 缓存 · `/downloads` 播放支持
